# distingNT_API/airwindows
Code to convert the [airwindows](https://github.com/airwindows/airwindows) plug-ins to the disting NT.

In `make_plugins`:
* `python3 make_main.py`
* `make bin`
* `make reports`
* `python3 make_plugins.py`

In `src`:
* `make`
